Arithmetic Progressions (AP)

Exercise 5.1 – Introduction to AP

Q1. Identify which of the following forms an AP:

(a) 2, 4, 8, 16 …
Differences: 2, 4, 8 → not equal → Not an AP.

(b) 5, 1, –3, –7…
Differences: –4, –4, –4 → equal → Yes, AP with d = –4.

(c) –1.5, –1.0, –0.5, 0 …
Differences: 0.5, 0.5, 0.5 → AP, d = 0.5.

(d) 1/3, 1/6, 0, –1/6…
Differences: –1/6, –1/6, –1/6 → AP with d = –1/6.

Q2. Write first 4 terms of AP when a and d are:

(a) a = 10, d = 10 → 10, 20, 30, 40
(b) a = –2, d = 0 → –2, –2, –2, –2
(c) a = 4, d = –3 → 4, 1, –2, –5

Exercise 5.2 – nth Term of an AP

Formula: aₙ = a + (n – 1)d

Q1. Find a₁₀ for AP: 2, 7, 12, 17...

a = 2, d = 5
a₁₀ = 2 + 9×5 = 47

Q2. Which term of AP: 3, 8, 13... is 78?

a = 3, d = 5
aₙ = 78
3 + (n – 1)5 = 78
(n – 1)5 = 75
n – 1 = 15 → n = 16th term

Q3. Is 0 a term of AP: 1, 4, 7, 10… ?

a = 1, d = 3
1 + (n–1)3 = 0
(n–1)3 = –1
n–1 = –1/3 → n = 2/3 → not an integer
So 0 is NOT a term.

Exercise 5.3 – Sum of First n Terms (Sₙ)

Formula:

Sₙ = n/2 [2a + (n–1)d]
or
Sₙ = n/2 (a + l)

Q1. Find S₁₀ of AP: 2, 5, 8, 11...

a = 2, d = 3
S₁₀ = 10/2[2×2 + 9×3]
= 5[4 + 27] = 5×31 = 155

Q2. Which term is the first negative term of AP: 11, 8, 5, 2...

a = 11, d = –3
Find n such that aₙ < 0
11 + (n–1)(–3) < 0
11 – 3n + 3 < 0
14 – 3n < 0
3n > 14 → n > 14/3 → n ≥ 5

So 5th term is first negative: a₅ = –1

Q3. Sum of first n terms of AP is 5n – n². Find AP.

Given Sₙ = 5n – n²
a₁ = S₁ = 5 – 1 = 4
a₂ = S₂ – S₁ = (10 – 4) – 4 = 2
a₃ = S₃ – S₂ = (15 – 9) – (10 – 4) = 0

AP is: 4, 2, 0 ...
d = –2

Exercise 5.4 – Applications of AP

Q1. Salary of a worker: ₹5000 first month; increases by ₹200 each month. Find the salary of the 15th month.

a = 5000, d = 200
a₁₅ = 5000 + 14×200 = 7800

Q2. A man saves ₹100 in first month, ₹120 in second, ₹140 in third… Find money saved in 12 months.

a = 100, d = 20, n = 12
S₁₂ = 12/2 [2×100 + 11×20]
= 6 [200 + 220] = 6×420 = 2520

Q3. A rope ladder has rungs 25 cm apart. Lowest rung is 1m from ground. If there are 20 rungs, what is height?

First rung height a = 1 m = 100 cm
Distance increases by 25 cm (d = 25)
20th rung height:
aₙ = 100 + (20 – 1)25
= 100 + 475 = 575 cm = 5.75 m

Summary – Arithmetic Progressions

  • AP: constant difference d = a₂ – a₁
  • nth term: aₙ = a + (n–1)d
  • Sum of n terms: Sₙ = n/2 [2a + (n–1)d]
  • If l = last term → Sₙ = n(a + l)/2
