Exercise 4.1 – Identifying Quadratic Equations

Q1. Which of the following are quadratic equations?

(a) x² + 3x + 2 = 0 → Yes
(b) x + 1/x = 2 → Multiply by x: x² - 2x + 1 = 0 → Yes
(c) x² + 2x = 5 → Rearranged: x² + 2x - 5 = 0 → Yes
(d) 2/(x²) + 1 = x → Multiply by x²: 2 + x² = x³ → cubic → No
(e) (x–2)(x–3) = x+6 → Expand: x² –5x +6 = x+6 → x² –6x = 0 → Yes

Q2. Convert to standard form ax² + bx + c = 0:

(a) x² = 3x + 1 → x² – 3x – 1 = 0
(b) 3x² – 2 = x → 3x² – x – 2 = 0
(c) x – 1/x = 3 → multiply by x: x² – 3x – 1 = 0

Exercise 4.2 – Solving by Factorisation

Q1. Solve by factorisation:

(a) x² + 7x + 10 = 0
    (x+5)(x+2) = 0
    x = –5, –2

(b) 2x² – 7x + 3 = 0
    2x² – 6x – x + 3 = 0
    (2x – 1)(x – 3) = 0
    x = 1/2, 3

(c) 3x² – x – 4 = 0
    3x² – 4x + 3x – 4 = 0
    (3x + 2)(x – 2) = 0
    x = –2/3, 2

Q2. Solve: (x – 3)(x + 2) = 4

Expand:
x² – x – 6 = 4
x² – x – 10 = 0
Factor:
(x – 5)(x + 2) = 0
x = 5, –2

Exercise 4.3 – Solving Using Quadratic Formula

Quadratic formula:

x = [-b ± √(b² – 4ac)] / 2a

Q1. Solve:

(a) 2x² – 7x + 3 = 0
    a=2, b=–7, c=3
    Δ = 49 – 24 = 25
    x = (7 ± 5)/4 = 3, 1/2

(b) x² – 3x – 10 = 0
    a=1, b=–3, c=–10
    Δ = 9 + 40 = 49
    x = (3 ± 7)/2 = 5, –2

(c) 2x² – 4x + 2 = 0
    Δ = 16 – 16 = 0
    One solution: x = 1

Q2. Solve (real or complex): x² + x + 1 = 0

a=1, b=1, c=1
Δ = 1 – 4 = –3 < 0
x = [-1 ± √3 i] / 2

Exercise 4.4 – Applications of Quadratic Equations

Q1. A train travels 300 km at a speed 10 km/h slower than return speed. Total time = 13h.

Let normal speed = x km/h
Return speed = x + 10

Time eqn:
300/x + 300/(x+10) = 13

Multiply:
300(x+10) + 300x = 13x(x+10)
300x + 3000 + 300x = 13x² + 130x

600x + 3000 = 13x² +130x
13x² – 470x – 3000 = 0
Solve:
x = 50 km/h
Return speed = 60 km/h

Q2. A number and its reciprocal differ by 7/6. Find the number.

Let number = x
x – 1/x = 7/6
Multiply by x:
x² – 7x/6 – 1 = 0
Multiply by 6:
6x² – 7x – 6 = 0
Solve:
(3x + 2)(2x – 3) = 0
x = –2/3, 3/2

Q3. Area of a rectangle is 192 cm². Length = 3 cm more than breadth.

Let breadth = x
Length = x + 3
x(x+3) = 192
x² + 3x – 192 = 0
Solve:
(x + 16)(x – 12) = 0

Positive solution: x = 12
Breadth = 12 cm
Length = 15 cm

Summary – Quadratic Equations

  • Standard form: ax² + bx + c = 0
  • Methods: Factorisation, Completing square, Quadratic formula
  • Discriminant Δ = b² – 4ac
  • Δ > 0 → two real solutions
  • Δ = 0 → one real solution
  • Δ < 0 → no real solution; complex roots
