Exercise 9.1 – Heights and Distances

Q1. Height of a tower using angle of elevation 30° and distance 50 m

tan θ = height / distance
tan 30° = h / 50 → h = 50 × tan 30° = 50 × 1/√3 ≈ 28.87 m

Q2. Distance of a tree if angle of elevation from top of building 45° and height 20 m

tan θ = height / distance
tan 45° = 20 / distance → distance = 20 m

Exercise 9.2 – Solving Problems Using Trigonometry

Q1. Ladder 10 m leaning against wall, foot 6 m from wall. Height reached?

Right triangle: Ladder = hypotenuse = 10 m, base = 6 m
Height = √(hypotenuse² – base²) = √(10² – 6²) = √(100–36) = √64 = 8 m

Q2. Angle of elevation of balloon changes from 30° to 60° as it rises 40 m

Let horizontal distance = x, height at lower position = h
tan 30° = h / x → h = x / √3
tan 60° = (h+40) / x → h+40 = x√3
x√3 – h = 40, h = x/√3 → x√3 – x/√3 = 40 → x(√3 – 1/√3) = 40
x(2/√3) = 40 → x = 20√3
h = x/√3 = 20√3 / √3 = 20 m
Height at higher point = 20 + 40 = 60 m

Exercise 9.3 – Applications in Real Life

Q1. Height of building from top of pole, angle of elevation 45°

Given angle θ = 45°, height of pole = h1, distance = d
tan θ = (height of building – h1) / d
Use formula to find building height = h1 + d × tan 45° = h1 + d

Q2. Distance of ship from cliff top using angles of depression

tan θ = height of cliff / distance from ship
Distance = height / tan θ
Use multiple angles if needed to find exact position

Summary – Some Applications of Trigonometry

  • Use right triangle relations: tan θ = opposite / adjacent
  • Height = distance × tan θ
  • Distance = height / tan θ
  • Use Pythagoras theorem for ladders, towers, etc.
  • Apply to angles of elevation and depression in real-life problems
